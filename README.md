```powershell
cargo +stable-x86_64-pc-windows-gnu run

# Open in browser:

```

Or use the build script:

```powershell

```

## Deploy to AWS

### 1. Prerequisites

- [AWS CLI](https://aws.amazon.com/cli/) configured (`aws configure`)
- [Terraform](https://developer.hashicorp.com/terraform/install) installed

### 2. Create AWS Infrastructure

```powershell
cd terraform

# Copy and edit variables
Copy-Item terraform.tfvars.example terraform.tfvars
# Edit terraform.tfvars with your bucket name

terraform init
terraform plan
terraform apply
```

Terraform outputs your CloudFront URL and distribution ID.

### 3. Deploy Site

```powershell
cd ..
.\scripts\deploy.ps1 -BucketName "your-bucket-name" -DistributionId "ABCDEF123"
```

This will:
1. Build the Rust site generator
2. Run it to generate `dist/`
3. Sync `dist/` to S3 with correct cache headers
4. Invalidate the CloudFront cache

Your site will be live at the CloudFront URL within ~5 minutes.

## Customization

All portfolio data (skills, experience, projects, certs) lives in `src/main.rs` as statically-typed Rust structs. Edit the `get_*` functions and re-run `cargo run` to regenerate.

The visual design is in `static/css/style.css` and the HTML structure in `templates/index.html`.
